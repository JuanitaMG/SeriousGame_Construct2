# Serious Game - Construct 2

## Descripción

Este repositorio contiene un videojuego 2D desarrollado en Construct 2 como parte de un proyecto académico durante la formación en Ingeniería en Informática.

El proyecto fue diseñado con el objetivo de aplicar conceptos relacionados con la lógica de programación mediante eventos, diseño de niveles, control de personajes y mecánicas básicas de un videojuego. La estructura del proyecto está organizada utilizando layouts, hojas de eventos y recursos gráficos propios de Construct 2.

## Características

- Movimiento del personaje.
- Sistema de colisiones.
- Recolección de puntos.
- Escenarios con obstáculos.
- Condiciones de victoria y derrota.
- Organización mediante Layouts y Event Sheets.
- Desarrollo orientado a la creación de videojuegos 2D.

## Tecnologías utilizadas

- Construct 2
- Event Sheets
- Layouts
- Animaciones 2D
- Sprites

## Estructura del proyecto

```text
Animations/
Event sheets/
Files/
Layouts/
Serious Game.caproj
```

## Cómo ejecutar el proyecto

1. Abrir Construct 2.
2. Seleccionar **File > Open**.
3. Abrir el archivo **Serious Game.caproj**.
4. Ejecutar el proyecto desde Construct 2 utilizando la opción **Run Layout**.

## Objetivo del proyecto

Desarrollar un videojuego funcional que permitiera fortalecer habilidades en diseño de niveles, programación mediante eventos, interacción entre objetos y organización de proyectos utilizando Construct 2.
