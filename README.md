Serious Game - Construct 2

Descripción



Este proyecto corresponde a un videojuego desarrollado en Construct 2 como parte de un trabajo académico.



El objetivo del juego es que el jugador avance a través del escenario evitando obstáculos y superando los diferentes desafíos hasta llegar a la meta.



El proyecto fue desarrollado utilizando eventos propios de Construct 2, organización por layouts y hojas de eventos para controlar la lógica del juego.
